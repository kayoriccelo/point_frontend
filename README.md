# Ponto Eletrônico
   Aplicação onde controle a entrada e saídas de cada funcionário de uma empresa.
    
   ### Instalação do projeto
   + Yarn 
      + Fonte de pesquisa
          + https://yarnpkg.com/pt-BR/docs/install#debian-stable

      + Instalando dependências:
          + sudo yarn install

      + Iniciando Serviço:
          + sudo yarn start
